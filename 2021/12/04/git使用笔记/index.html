上传

  1. ```
    git add .

  2. git push -u origin [分支名]

git checkout -b 分支名

git checkout 分支名 #可以切换到该分支

git checkout 分支名 #可以切换到该分支

2.将修改的内容上传至自己的分支

  1. git add .

  2. git commit -m "说明"

  3. 分两种情况

    1. github上已有你的分支
      git push -u origin 分支名

    2. github上没有你的分支，需要在远程创建与本地对应的分支

      git push --set-upstream origin 分支名

3.合并 Merge

1.首先在git中将分支切换到main

git checkout main

2.拉去一下当前项目的更新

git pull

意思就是先将远程仓库里的版本与自己本地仓库的main分支进行合并（因为别人可能修改了）；再将自己修改的内容合并上去。

3.合并

git merge 你的分支名

4.再push一次

git push

pull拉取是直接拉取到本地仓库

如果pull超时，且你在科学上网。就先关掉vpn再拉取。
